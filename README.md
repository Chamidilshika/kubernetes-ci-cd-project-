# Kubernetes CI/CD Pipeline Project 🚀

## 📌 Project Overview

This project demonstrates a complete CI/CD pipeline implementation using Docker, Kubernetes, Minikube, Jenkins, and GitHub.

The application is containerized using Docker and deployed into a Kubernetes cluster running on Minikube. Jenkins is integrated to automate the CI/CD workflow.

## 🛠 Technologies Used

* Docker
  
* Kubernetes
  
* Minikube
  
* Jenkins
  
* GitHub
  
* Node.js
  
* VS Code




## ⚙️ Prerequisites

Before running this project, install the following:

* Docker Desktop
  
* Minikube
  
* kubectl
  
* Git
  
* Jenkins (Docker container)
  
* VS Code
